# Конструктор карточек

Веб-приложение для пакетного создания карточек постов 1000x1000.

## Запуск

```bash
npm install
npm start
```

Откройте адрес из терминала на компьютере. Чтобы открыть на телефоне в той же сети, используйте Network URL, который показывает Vite.

## Сборка для передачи или публикации

```bash
npm run build
```

Готовые статические файлы появятся в папке `build`. Ее можно опубликовать на любом статическом хостинге.

## Шаблоны и шрифт

- Каталог групп и шаблонов: `public/templates/templates.json`
- SVG-шаблоны: `public/templates/...`
- Шрифт: `public/fonts/stolzl_medium.otf`

SVG-шаблоны используются как оверлей поверх фотографии. Текстовые поля, координаты, ширина, угол поворота, стили и размер карточки задаются в `templates.json`. Соотношение сторон карточки берется из `width` и `height`, которые офлайн-редактор считывает из SVG.

## Офлайн-редактор шаблонов

Откройте файл `template-editor/index.html` в браузере. В нем можно создать группы, шаблоны и текстовые поля, затем скачать новый `templates.json` и положить его в `public/templates/templates.json`.
