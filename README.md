# Конструктор карточек

Веб-приложение для пакетного создания карточек постов 1000x1000.

## Запуск

```bash
npm install
npm start
```

Откройте адрес из терминала на компьютере. Чтобы открыть на телефоне в той же сети, используйте Network URL, который показывает Vite.

## Сборка для передачи или публикации

```bash
npm run build
```

Готовые статические файлы появятся в папке `build`. Ее можно опубликовать на любом статическом хостинге.

## Шаблоны и шрифт

- Шаблон с заголовком: `public/templates/template-with-title.svg`
- Шаблон без заголовка: `public/templates/template-no-title.svg`
- Шрифт: `public/fonts/stolzl_medium.otf`

SVG-шаблоны используются как оверлей поверх фотографии. Текст рисует приложение отдельно, чтобы его можно было менять для каждой карточки.
