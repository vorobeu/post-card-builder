# Редактор шаблонов

Откройте `index.html` в браузере. Редактор работает локально и не требует установки.

## Как пользоваться

1. Создайте группы и шаблоны.
2. В поле `Папка внутри /templates` укажите подпапку, например `moscow`.
3. Нажмите `Выбрать SVG`; редактор сам подставит путь вида `/templates/moscow/title-1.svg` и считает размер из SVG.
4. Добавьте текстовые поля и расположите их мышью или числовыми настройками.
5. Нажмите `Скачать templates.json`.
6. Положите файл в `public/templates/templates.json`.

SVG-файлы должны лежать в `public/templates/...`.
